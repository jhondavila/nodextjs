Descripción de DevKit

DevKit es un conjunto de herramientas que facilita la creación y el uso de extensiones C / C ++ nativas, como RDiscount y RedCloth para Ruby en Windows.

Simplemente descargue, haga doble clic, elija un directorio de instalación, ejecute el script Ruby install helper, posiblemente modifique un config.ymlarchivo para las especificaciones de su sistema, y ??estará listo para comenzar a usar las extensiones nativas de Ruby.
